# Spreewölfe Berlin — Community Match Calendars

> **⚠️ This is an unofficial community project.**
> It is **not** created by, affiliated with, or endorsed by
> [Spreewölfe Berlin e.V.](https://www.spreewoelfe.de/)
> All schedule data is sourced from publicly available information on
> [bishl.de](https://www.bishl.de/) (Berliner Inline-Skater-Hockey-Liga).
> For official information, always refer to the club's website.

This project provides iCalendar (`.ics`) files for the
[Spreewölfe Berlin](https://www.spreewoelfe.de/) inline skaterhockey
match schedules — **only Spreewölfe games** — so that fans, parents,
and players can easily subscribe to game dates in their calendar apps.

## How to Generate the Calendar Files

The iCal files are generated by scraping the schedule pages on
[bishl.de](https://www.bishl.de/). The script filters for Spreewölfe
games only. Re-run once per year when new schedules are published.

### Prerequisites

```bash
pip install requests beautifulsoup4
```

### Generate All Known Tournaments

```bash
cd spreewoelfe/
python fetch_schedules.py --year 2026 --output .
```

### Generate a Single Tournament

```bash
python fetch_schedules.py --tournament schuelerliga-lk2 --year 2026 --output .
```

### Options

```text
--tournament SLUG  Fetch only one tournament (e.g. schuelerliga-lk2)
--year YEAR        Season year (default: current year)
--output DIR       Output directory (default: current directory)
```

### Adding New Tournaments

Edit `KNOWN_TOURNAMENTS` in `fetch_schedules.py` to add more BISHL
tournament slugs. The slug is the part after `/tournaments/` in the
bishl.de URL.

## Locations Database

The file [`locations.json`](locations.json) contains a database of
all known venue addresses. It is updated automatically when the fetch
script runs. You can also edit it manually.

## Yearly Checklist

This project is intended to be updated once per season:

1. Wait for schedules to be published on
   [bishl.de](https://www.bishl.de/) (usually early in the year)
2. Check for new tournament slugs and add them to `KNOWN_TOURNAMENTS`
3. Run `python fetch_schedules.py --year YYYY --output .`
4. Verify the generated `.ics` files (open them in a calendar app)
5. Update this README: change the year numbers and QR code URLs
6. Commit and push
